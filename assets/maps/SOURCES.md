Bundled test packs in this folder were repacked from these free GitHub repositories:

- `oacmpvol1.pk3`: https://github.com/NeonKnightOA/oacmpvol1
- `oacmpvol2.pk3`: https://github.com/NeonKnightOA/oacmpvol2

Both repositories are published under `GPL-2.0` and contain Quake 3 / OpenArena-compatible map content.

The `.pk3` files here were created by zipping each repository's `baseoa/` directory contents directly into the archive root.
